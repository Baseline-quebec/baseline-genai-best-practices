## [1.0.0] - 2026-04-17

### Added
- Four main sections: Prompt Engineering, Governance & Ethics, Agents & Tools, Skills
- Skill library with two meta-skills: skill_builder and skill_auditor
- Skill specification format with YAML frontmatter and progressive disclosure model
- Connector design principles, specification format, and red flags
- Context management guidelines and context budget pattern
- Team skill sharing conventions: naming, ownership, versioning, and lifecycle
- Contributing guidelines with style guide and document structure template
- Code of Conduct (Contributor Covenant 2.1)
- GitHub PR template and issue templates (pattern proposal, correction)
- Version badge and Code of Conduct badge in README
- Table of contents in skill library README files
